Tentang Smart Tag Manager

Smart Tag Manager adalah aplikasi web sederhana namun powerful untuk mengelola tag Anda. Dibuat dengan teknologi web modern, aplikasi ini menyimpan tag Anda di browser Anda sendiri, sehingga tidak perlu khawatir kehilangan data.

Fitur:

  • Tambah tag dengan cepat
  • Edit dan hapus tag secara real-time
  • Simpan tag ke localStorage
  • Tampilan modern dan futuristik
  • Desain responsif

Teknologi:

  • HTML5
  • CSS3 (dengan efek glitch & gradient)
  • JavaScript (DOM manipulation & localStorage)
  • Particles.js (untuk efek latar)
